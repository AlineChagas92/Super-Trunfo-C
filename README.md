Super Trunfo em C

Projeto desenvolvido em linguagem C para o desafio SUPER TRUNFO.
O trabalho possui três níveis de complexidade: Novato, Aventureiro e Mestre; demonstrando a evolução no uso de estruturas de decisão, menus interativos e lógica de comparação.

Estrutura do Projeto

- nivel_novato.c
  Comparação simples entre duas cartas com atributo fixo definido no código, utilizando estruturas if e if-else.

- nivel_aventureiro.c
  Implementa um menu interativo com switch, permitindo ao usuário escolher um atributo para comparação entre duas cartas.

- nivel_mestre.c 
  Versão final e mais completa do projeto, com escolha de dois atributos diferentes, menus dinâmicos, regras específicas de comparação, soma dos atributos, tratamento de empates e uso do operador ternário.

Funcionalidades por Nível

Nível Novato:
- Duas cartas pré-cadastradas
- Cálculo de densidade demográfica e PIB per capita
- Comparação de um único atributo definido no código
- Uso de if / else
- Exibição clara do vencedor ou empate

Nível Aventureiro:
- Menu interativo para escolha do atributo
- Uso de switch e if-else aninhados
- Comparação entre duas cartas
- Regra invertida para densidade demográfica
- Tratamento de empate e opção inválida

Nível Mestre
- Escolha de dois atributos numéricos diferentes
- Menus dinâmicos (não permite repetir atributo)
- Comparação individual de cada atributo com regra adequada
- Soma dos atributos vencedores
- Tratamento de empates
- Uso de switch, if-else, default e operador ternário
- Exibição detalhada do resultado final
